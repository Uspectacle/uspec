## 🚀 Getting Started

### Prerequisites

- Node.js >= 24.0.0
- npm >= 11.0.0

### Install & Run

```bash
git clone https://github.com/Uspectacle/uspec
cd uspec
pnpm install
pnpm start
```

Site runs at `http://localhost:3000`

## 🔧 Scripts

```bash
pnpm start           # Dev server
pnpm build           # Static export
pnpm lint            # Lint
pnpm lint-and-fix    # Lint + fix
pnpm prettier-format # Format
```
